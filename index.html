📝 Challenge Lógica de Programación - Lista de Tareas

Este proyecto fue desarrollado como parte del Challenge de Lógica de Programación propuesto por Alura Latam y Oracle Next Education (ONE).

🚀 Descripción del Proyecto

Se desarrolló una aplicación de consola en Python que permite gestionar una lista de tareas (To-Do List). El usuario puede:

Agregar nuevas tareas. Ver todas las tareas pendientes. Eliminar tareas completadas. Salir del programa. Todo esto se logra aplicando los principios básicos de programación estructurada.
